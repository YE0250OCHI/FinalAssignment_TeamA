# チームA 最終課題：自動倉庫システム

## 概要

本プロジェクトでは、以下のプロダクトを開発する

- 複数台の自動倉庫を制御・管理するためのコントローラ（APIサーバー）
- 自動倉庫単体のコントローラ（ピッキング機器スタブ）
- ピッキング指示出し端末（PWA）

### 背景

A社では、社内工場で部品製造から組み立てまでを一貫して行い、完成品を出荷している。その工場では、各部品を製造後、一度エレクターに保管しておき、組み立てライン側で必要になったタイミングで、人が都度エレクターから部品をピッキングし、指定された組み立てラインまで搬送していた。昨今の人手不足問題による、配置転換のため、このピッキング要員を減らすべく、自動倉庫システムを導入することとなった。本プロジェクトでは、この自動倉庫群を制御・管理するためのコントローラ、自動倉庫単体のコントローラおよび、操作画面を開発する。
